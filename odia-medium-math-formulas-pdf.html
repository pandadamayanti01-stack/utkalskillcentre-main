ଶ୍ରେଣୀ (Class): ଶିଶୁବାଟିକା ୧ମ ୨ମ ୩ୟ ୪ର୍ଥ ୫ମ ୬ଷ୍ଠ ୭ମ ୮ମ ୯ମ ୧୦ମ
ଗଣିତ ସୂତ୍ର ଚିଟ୍ ସିଟ୍

ଓଡ଼ିଆ ମାଧ୍ୟମ ହାଇସ୍କୁଲ ଗଣିତ ସୂତ୍ରାବଳୀ (Class 8-10)

ଓଡ଼ିଶା ବୋର୍ଡ ହାଇସ୍କୁଲ (ଶ୍ରେଣୀ ୮, ୯ ଓ ୧୦) ଗଣିତ ବିଷୟର ସମସ୍ତ ଗୁରୁତ୍ୱପୂର୍ଣ୍ଣ ସୂତ୍ରଗୁଡ଼ିକୁ ଏଠାରେ ସହଜରେ ପାଆନ୍ତୁ। ଆମର AI ଟ୍ୟୁଟର ଗୁଣ୍ଡୁଲୁ ସାହାଯ୍ୟରେ ସୂତ୍ରଗୁଡ଼ିକର ବ୍ୟବହାର ଓ ସମାଧାନ ପ୍ରକ୍ରିୟା ଶିଖନ୍ତୁ।

ଗେଷ୍ଟ ଜୋନ୍ ପ୍ରବେଶ କରନ୍ତୁ

ଅଧ୍ୟାୟ ଭିତ୍ତିକ ମୁଖ୍ୟ ସୂତ୍ରାବଳୀ

📈 ସମାନ୍ତର ପ୍ରଗତି (Arithmetic Progression)

ସମାନ୍ତର ପ୍ରଗତିର nth ପଦ (term) ନିରୂପଣ କରିବାର ସୂତ୍ର:

tn = a + (n - 1)d

ପ୍ରଥମ n ସଂଖ୍ୟକ ପଦର ସମଷ୍ଟି (Sum):

Sn = n/2 [2a + (n - 1)d]

📐 ଦ୍ଵିଘାତ ସମୀକରଣ (Quadratic Equations)

ax² + bx + c = 0 ସମୀକରଣର ମୂଳ (Roots) ନିରୂପଣ ସୂତ୍ର:

x = [-b ± √(b² - 4ac)] / 2a

ପ୍ରଭେଦକ (Discriminant) D = b² - 4ac

🌐 ସ୍ଥାନାଙ୍କ ଜ୍ୟାମିତି (Coordinate Geometry)

ଦୁଇଟି ବିନ୍ଦୁ (x₁, y₁) ଏବଂ (x₂, y₂) ମଧ୍ୟରେ ଦୂରତା (Distance) ସୂତ୍ର:

d = √[(x₂ - x₁)² + (y₂ - y₁)²]

📐 ପରିମିତି (Mensuration - Cylinder)

ସିଲିଣ୍ଡରର ବକ୍ର ପୃଷ୍ଠତଳର କ୍ଷେତ୍ରଫଳ (Curved Surface Area):

A = 2πrh

ସିଲିଣ୍ଡରର ଆୟତନ (Volume):

V = πr²h
